# PyMeshGen GUI 使用说明

## 简介

PyMeshGen GUI 是 PyMeshGen 网格生成器的图形用户界面版本，提供了直观的参数设置、文件导入导出、网格显示和交互功能。

## 功能特性

1. **参数设置**
   - 调试级别设置
   - 输入/输出文件路径设置
   - 网格类型选择
   - 可视化开关

2. **文件操作**
   - 导入 Fluent CAS 格式网格文件
   - 导出 VTK 格式网格文件
   - 配置文件的加载和保存

3. **网格显示**
   - 2D 网格结构可视化
   - 边界类型颜色区分
   - 交互式图形显示

4. **网格生成**
   - 一键生成网格
   - 进度显示

## 安装依赖

确保已安装项目所需的所有依赖包：

```bash
pip install -r requirements.txt
```

## 运行 GUI

在项目根目录下运行以下命令启动 GUI：

```bash
python gui/gui_main.py
```

## 使用方法

### 1. 参数设置
- 在参数设置区域配置网格生成所需的各种参数
- 可通过"浏览"按钮选择输入输出文件路径
- 选择合适的网格类型（三角形、直角三角形或混合网格）

### 2. 文件操作
- 使用菜单栏的"文件"选项进行配置文件和网格文件的导入导出
- 支持 JSON 格式的配置文件
- 支持 Fluent CAS 格式的输入和 VTK 格式的输出

### 3. 网格显示
- 导入网格文件后，点击"显示网格"按钮查看网格结构
- 不同类型的边界会以不同颜色显示
- 可通过"清除显示"按钮关闭图形窗口

### 4. 网格生成
- 配置好参数后，点击"生成网格"按钮开始网格生成过程
- 生成的网格将自动保存到指定的输出文件

## 界面说明

### 菜单栏
- **文件**: 配置文件和网格文件的导入导出操作
- **视图**: 网格显示相关操作
- **帮助**: 关于信息

### 参数设置区域
包含所有网格生成相关的参数配置选项

### 控制按钮区域
提供网格生成、显示和清除等快捷操作按钮

### 状态栏
显示当前操作状态和进度信息

## 开发说明

GUI 主程序位于 `gui/gui_main.py`，基于 tkinter 开发。

主要组件：
- `PyMeshGenGUI`: 主 GUI 类
- 参数管理: 与核心库的 Parameters 类集成
- 文件操作: 与 fileIO 模块集成
- 可视化: 与 visualization 模块集成

## 注意事项

1. 确保已正确安装所有依赖包
2. 输入文件应为 Fluent CAS 格式
3. 输出文件为 VTK 格式，可用 Paraview 等软件查看
4. 网格生成功能需要进一步完善核心算法集成
